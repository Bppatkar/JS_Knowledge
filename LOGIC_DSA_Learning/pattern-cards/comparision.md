# Comparison Pattern

## Recognition

Compare the current element against a condition while traversing.

## Visualization

A security guard checks each visitor against entry rules.

## Engineering Analogy

* Age validation
* Permission checking
* Score comparison
* Threshold monitoring

## Generic Algorithm

Traverse.

Compare current element.

Take action if condition is true.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Common Bugs

* Wrong comparison operator
* Comparing incorrect variables
* Incorrect logical conditions

## Related Patterns

* Early Exit
* Maximum Pattern
* Minimum Pattern

Compare

## Interview Tip

Every if statement inside a traversal is usually a comparison pattern.
