# Count Pattern

## Recognition

Count how many elements satisfy a condition.

## Visualization

A teacher counts how many students are present in a classroom.

## Engineering Analogy

* Number of active users
* Successful API requests
* Failed logins
* Sold products

## Generic Algorithm

Initialize count to zero.

Traverse the array.

If condition is true

Increment count.

Return count.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Common Bugs

* Forgetting count initialization
* Incrementing in the wrong place
* Incorrect condition

* Running Count
