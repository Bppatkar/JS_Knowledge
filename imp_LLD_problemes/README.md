# Important LLD Problems

### Defination - Low level design problems are practical system exercises that teach how to split one big problem into smaller classes and responsibilities.

#### (Hindi Meaning) - LLD ka matlab hota hai problem ko classes, objects, aur clear responsibilities me todna.

### How to study

1. Defination padho.
2. Problem samjho.
3. Solution dekho.
4. Wrong code dekho.
5. Fixed code samjho.
6. Same school example ko yaad rakho.
